# tsoha
## Tietokantasovellus - aineopin harjoitustyö

### Aihe: Keskustelusovellus

- Foorumityyppinen keskustelusovellus, jossa käyttäjät voivat osallistua viestikeskusteluihin eri aihealueilla. Käyttäjillä on myös mahdollisuus lähettää toisilleen yksityisviestejä.
- Käyttäjät voivat luoda tunnuksen sekä kirjautua sisään ja ulos. Käyttäjät ovat joko peruskäyttäjiä tai ylläpitäjiä.
- Sovelluksen etusivulla on lista aihealueista; alueet voidaan järjestää aakkosjärjestyksessä, laajusjärjestyksessä (alueen viestien määrä) tai tuoreusjärjestyksessä (viimeksi muokatut).
- Käyttäjät voivat luoda uusia viestiketjuja aihealueille ja poistaa luomiaan ketjuja. Ylläpitäjät voivat poistaa ketjuja, luoda uusia alueita ja jäädyttää peruskäyttäjiä.
- Käyttäjät voivat muokata viestejään ja vastata olemassa oleviin ketjuihin.
- Käyttäjät voivat hakea viestejä, ketjuja tai aihealueita.
- Ylläpitäjät voivat siirtää ketjuja aihealueiden välillä ja käyttäjät voivat jättää palautetta, pyyntöjä tai ehdotuksia ylläpitäjille.
- Ylläpitäjät voivat rajoittaa käyttäjien pääsyä alueille ja luoda salaisia alueita.
- Käyttäjät voivat hakea ja lisätä toisia käyttäjiä kontaktiluetteloon ja lähettää heille yksityisviestejä. 
- Käyttäjät voivat hakea toisia käyttäjiä tai viestejä kontakteistaan.
